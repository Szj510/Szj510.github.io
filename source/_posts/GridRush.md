# 我最近用AI做了个小游戏，叫 GridRush

我最近用 AI 做了个小游戏：GridRush，这是一个基于 React + TypeScript 的在线策略对战项目，把 3×3 棋盘博弈和快节奏微型小游戏结合在一起，支持禁选、技能对抗、抢格与防守等机制，整体节奏很快但也有策略深度。

如果你想进一步了解实现细节和代码结构，可以直接看 GitHub 源码：

- https://github.com/Szj510/GridRush

如果你想先体验玩法，也欢迎直接试玩线上版本：

- https://gridrush.pages.dev

欢迎提 issue、提 PR，或者直接在线上版本处提交feedback。
